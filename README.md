# Helocobic-energy

## 心血来潮，突然想做一下稚晖君的小电视
## 一、设计思路
 主控：STM32F411（起初使用401，但资料过少放弃）+ E32 双主控（E32支持蓝牙）
 外设：CP2102串口芯片，1.3寸TFT，MPU6050，TF卡
